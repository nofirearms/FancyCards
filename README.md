# FancyCards


## Audio Flashcards

Audio Flashcards is a small tool for learning through **audio-based flashcards** and **spaced repetition**.

Each card is built around a short audio sample. Along with the audio, you can store the original text, transcription, translation, and any personal notes. The app schedules reviews using spaced repetition to help retain material over time.

### Features

* Audio-centered flashcards

  * record or import audio
  * trim short segments for focused listening
* Rich card content

  * source text
  * transcription
  * translation
  * custom notes
* Spaced repetition system

  * cards are shown when they are due for review
  * intervals adapt based on recall
* Listening-first approach

  * audio before text
  * suitable for training speech perception
* Card collections

  * organize cards by language, topic, or source

### Use cases

* language learning from real audio (podcasts, videos, dialogs)
* shadowing and pronunciation practice
* vocabulary and phrase acquisition with context

### Philosophy

The project focuses on **real audio fragments** rather than isolated words.
Learning is based on repeated exposure and recognition, especially by ear.
